# eCNH API Roadmap

This repository currently provides the API foundation. Recommended next milestones:

## Near term

- Replace placeholder network contract addresses with audited deployment metadata.
- Add authenticated admin endpoints for maintaining network configuration.
- Add deployment manifests for the selected production platform.
- Add OpenAPI examples for partner onboarding.

## Integrations

- Connect quote generation to an approved treasury/oracle source once governance requirements are defined.
- Add partner API keys, request signing, and rate limiting.
- Add observability: structured logs, metrics, and trace IDs.

## Quality and compliance

- Add end-to-end tests against a staging deployment.
- Define incident response and network pause procedures.
- Add data retention and compliance documentation for production use.
